# Sentiment Analysis using Natural Language Processing (NLP)

## CodeAlpha Data Analytics Internship Project

**Author:** Harshit

---

## Project Overview

This project demonstrates the application of Natural Language Processing (NLP) techniques to analyze textual data and determine the sentiment expressed in reviews. The goal is to classify reviews into Positive, Negative, and Neutral categories using sentiment polarity scores generated by TextBlob.

Sentiment Analysis is widely used in industries to understand customer opinions, monitor brand reputation, analyze product feedback, and support data-driven decision-making.

---

## Objectives

* Perform sentiment analysis on textual reviews.
* Classify reviews into Positive, Negative, and Neutral sentiments.
* Visualize sentiment distribution using graphical representations.
* Understand how NLP techniques can extract meaningful insights from text data.

---

## Technologies Used

* Python
* TextBlob
* Pandas
* Matplotlib
* Jupyter Notebook

---

## Methodology

### 1. Data Collection

A collection of text reviews was used as input data.

### 2. Sentiment Processing

TextBlob was used to calculate the sentiment polarity of each review.

### 3. Classification

Based on polarity scores, reviews were categorized into:

* Positive
* Negative
* Neutral

### 4. Visualization

A pie chart was created to represent the sentiment distribution and provide an intuitive understanding of the results.

---

## Project Outputs

* Sentiment Classification Table
* Positive, Negative, and Neutral Review Counts
* Sentiment Distribution Pie Chart

---

## Key Insights

* Positive reviews exhibited higher sentiment polarity scores.
* Negative reviews showed lower sentiment polarity values.
* Sentiment analysis successfully categorized textual opinions into meaningful groups.
* Visualization made sentiment trends easier to interpret and communicate.
